# Core Blocks

Core Blocks for the WordPress editor.

## Installation

Install the module

```bash
npm install @wordpress/block-library --save
```

## Usage

```js
import { registerCoreBlocks } from '@wordpress/block-library';

registerCoreBlocks();
```
